Broken Access Control

Testăm problemele de control al accesului care permit utilizatorilor neautorizați să acceseze resurse restricționate.

  • Escaladarea privilegiilor – Un utilizator obișnuit poate obține acces administrativ.
  • Acces neautorizat la conturi (IDOR) – Posibilitatea de a accesa contul altui utilizator prin schimbarea unui ID.
  • Lipsa restricțiilor pe acțiuni importante – Permite ștergerea sau modificarea datelor altor utilizatori.

Cross-Site Scripting (XSS)

Explorăm atacurile XSS, unde input-ul nesigur este injectat în pagină pentru a executa scripturi malițioase.

  • Stored XSS – Codul malițios este salvat permanent în baza de date.
  • Reflected XSS – Atacatorul introduce cod malițios printr-o solicitare HTTP.
  • DOM-based XSS – Manipularea nesigură a DOM-ului permite rularea de cod JavaScript injectat.

Broken Authentication

Demonstrează problemele legate de autentificare care pot permite atacatorilor să compromită conturi utilizator.

  • Parole slabe – Permite atacuri brute-force.
  • Stocarea sesiunii în client-side nesecurizat.
  • Lipsa rate limiting la login.

Cryptographic Failures + CSRF

Demonstrează problemele legate de criptografie care pot duce la expunerea datelor sensibile.

  • Utilizarea algoritmului MD5 – Vulnerabil la atacuri rainbow table.
  • Hardcoding de chei criptografice – Un atacator poate extrage cheia.
  • Atac CSRF prin schimbarea adresei de e-mail.

Server-Side Request Forgery (SSRF)

Explorăm atacurile SSRF, unde utilizatorul poate introduce o adresă URL nesigură.

  • Manipularea cererilor către backend.
  • Acces la resurse interne.
  • Expunerea informațiilor sensibile.

Path Traversal

Explorăm atacul Path Traversal, unde un atacator poate accesa fișiere interne ale serverului.

  • Citirea fișierelor sensibile.
  • Exploatarea fără protecție.
  • Accesarea fișierelor interne.

Remote Code Execution (RCE)

Explorăm atacurile RCE, unde un atacator poate executa comenzi pe server.

  • Executarea comenzilor la distanță.
  • Injecție de comenzi.
  • Escaladarea privilegiilor.
